# Bolivia: fragilidad del poder y democracia | 1825–2025

Página estática de visualización histórica sobre la fragilidad del poder político, los ciclos de acceso y salida del Ejecutivo, la tensión entre gobiernos civiles y militares, y la evolución de la calidad democrática en Bolivia entre 1825 y 2025.

## Contenido del repositorio

- `index.html`: landing page completa, con estilos internos y visualizaciones embebidas.
- `.nojekyll`: archivo vacío para evitar procesamiento con Jekyll en GitHub Pages.

## Visualizaciones externas

La página usa visualizaciones públicas embebidas desde:

- Flourish
- Datawrapper

Para que la publicación funcione correctamente, las visualizaciones deben permanecer públicas o accesibles desde sus respectivos enlaces de inserción.

## Publicación en GitHub Pages

1. Crear un repositorio nuevo en GitHub, por ejemplo: `bolivia-fragilidad-democracia`.
2. Subir `index.html`, `README.md` y `.nojekyll` a la raíz del repositorio.
3. Entrar en **Settings → Pages**.
4. En **Build and deployment**, seleccionar:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guardar la configuración.
6. GitHub generará una URL pública del tipo:
   `https://usuario.github.io/bolivia-fragilidad-democracia/`

## Recomendaciones antes de publicar

- Verificar que todos los gráficos de Flourish y Datawrapper estén publicados.
- Revisar títulos, descripciones y fuentes antes de difusión pública.
- Agregar una imagen `og-image.png` si se quiere mejorar la vista previa en redes sociales.
- Si se agregan archivos locales en el futuro, crear carpetas como `assets/`, `css/` o `js/`.
